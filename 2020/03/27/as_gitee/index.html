现在正式讲此文内容

进入AndroidStudio点击 VCS -> Import into Version Control -> Create Git Repository，然后选择当前项目目录点右下角OK。

创建好Git仓库后那怎么上传到Gitee上呢？
点击VCS -> Commit
在Commit Message 信息栏中输入Tag信息，例如：新版本发布。

这时候你可能会发现Commit是灰色的不能点击，我们只要选择你将要上传代码的文件夹就可以了，点击树状文件夹列表顶部文件夹左边的多选框，他会自动全部选择下面的文件夹，如图。

最后点击右下角的Commit即可
将文件添加到待提交列表(缓存)中后，我们下面开始提交的教程吧。
点击VCS->Git->Push

点击上面的 Define remote 创建一个远程服务器，然后在URL框中输入你复制的URL，例如：https://gitee.com/kaodan/qk-php-sdk.git.

点击OK得到,最后，点击右下角Push，就可以上传服务器啦。

如果要改上传分支的话，可以双击master修改。

总结

你以后要在此项目上传的话，可以直接像下面这样操作。
先VCS -> Commit（将文件缓存到待提交列表中）
再VCS -> Git -> Push（将列表中的文件推送到Gitee上）
